# Tachometer ESP8266 

### Conponent
- ESP8266 (NodeMCU)
- Display SSD1306
- IR Infrared sensor

### PIN
- DISPLAY: GPIO4, GPIO5 (NodeMCU - D1, D2)
- SENSOR: GPIO0 (NodeMCU - D3)

### Specification .bin files

- Module: Generic ESP8266 Module
- Flash Size: 1MByte
- Flash Mode: QIO
- Flash Frequency: 40Mhz
- Upload Speed: 921600
